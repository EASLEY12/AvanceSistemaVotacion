Requerimientos
Diagaramas
Registro de candidatos y partidos.
Este proceso es básico para poder desarrollar todas las funciones adicionales de este sistema. Por ello hemos tomado a bien que atraves de la interfaz se pueda guardar la información que se procesaran a la base de datos, siendo así la entrada básica de datos.

Creación de papeleta de votación.
Los formularios creados automáticamente para la papeleta de votación deberán poder controlar los datos que estos procesaran y poder guardarlos en la base de datos. Es importante mencionar que en este modelo es de adaptar los datos a mostrar de acuerdo a los tipos de elección que se realicen.

Mostrar resultados finales
Este será un elemento de salida básica de información y así los usuarios y ciudadanos podrán visualizar los movimientos o estados de resultados de porcentajes de votos atraves de una grafica en tiempo real
